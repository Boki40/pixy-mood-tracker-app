### Development

**Setup**

1. Clone the repo
```shell
$ git clone https://github.com/mrzmyr/pixy-mood-tracker.git
```
2. Install dependencies
```shell
$ yarn
```
3. Start local server
```shell
$ yarn start
```

**Release Version**

```
$ npx standard-version --release-as [major|minor|patch]
```
